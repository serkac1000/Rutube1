About This Guide

This documentation provides a comprehensive troubleshooting guide for developers migrating from the PersonalImageClassifier extension to the TeachableMachine.aix extension in MIT App Inventor.

If you're experiencing issues with your migrated app, particularly with the TeachableMachine extension not starting the video classifier, this guide will help you identify and resolve common problems.

Common Migration Issues

When migrating from PersonalImageClassifier to TeachableMachine extension, developers often encounter these issues:

  • Video classifier fails to start
  • Incorrect initialization sequence
  • Missing permissions
  • Different component properties and methods
  • Incompatible event handlers
  • Classification results not being captured
